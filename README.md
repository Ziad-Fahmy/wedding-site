# Nora & Maged — Wedding Invitation Site (React)

A beautiful wedding invitation website built with React + Vite.

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v18 or higher (includes npm)

### Setup

1. **Unzip** this folder somewhere on your computer.

2. **Open a terminal** inside the `nora-maged` folder.

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. Open your browser at **http://localhost:5173**
